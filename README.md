# News Search app

This project is simple web app in which we can fetch the current news and display it.

/Features/

Search the news article by one keyword.
Dynamically fetch the images related to the news you searched.
Accessible for all the age groups.
Modern and professional layout similar like trending one's.

/languages used in this project./

*HTML for structure
*Css for the layout
*Javascript for async logics to perform the tasks.
*NEWS_API for real time data.
